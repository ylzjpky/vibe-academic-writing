# Examples

Every example in this directory must be synthetic and reusable without an institution account. Do not contribute real assignment briefs, course names, course files, student work, LMS URLs, identifiers, or screenshots.

## Single task

```text
Use develop-academic-paper for my 1,500-word international-relations essay. The attached brief sets APA 7 and a two-week deadline. Extract its requirements and ask me only for information that blocks the next step. Use public scholarly sources.
```

## Course synchronization

```text
Use develop-academic-paper in course mode for the selected fictional course. Synchronize currently available non-video materials. I will handle login and MFA in the website. Compare the remote and local inventories and report anything unavailable without retrying access failures.
```
